Event subacriptions are designed to receive a particular resource (such as an order or shipment). You can supply a comma separated list of child resources you wish to expand for use within your template. For example, if you are creating an order confirmation event subscription, your event subscription will receive the order object and you way wish to expand the customer resource.

To increase performance of event subscription rendering, you can limit the fields that are supplied to your event subscription to only the ones your event subscription needs. Read more about customizing resources. If you supply a value for show, you cannot supply a value for hide.
